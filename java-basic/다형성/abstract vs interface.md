# 자바가 전하는 abstract VS interface

abstract :상속받은 클래스야. 내 바디에 있는 method 들. 구현체 꼭 다 만들 필요는 없고, 쓸 것만 갖다가 써라. override는 꼭 하고 써라
---
---

interface  : 상속 받은 클래스야. 내 바디에 있는 method들, override 꼭 해서, 구현체 전부 만들어라. 안만들면 프로그램 실행 못하게 할거임.
